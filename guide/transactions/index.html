Guide

One input change becomes one deterministic transaction result.

The host stages changes. The graph computes derived state, diffs, plans, output frames, revisions, and audit entries. The host applies returned commands after propagation.

Stageinput and scope changes
Validatereferences and cycles
Derivedirty nodes
Diffcollections
Planresource commands
Commitrevision and frames

Graph phase

No external I/O, task spawning, host callbacks, or resource handle mutation.

Host phase

The actor applies resource plans, emits output frames, and reports later resource status as canonical input.
