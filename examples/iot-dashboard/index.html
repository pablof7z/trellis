Problem

Trace

Before {pump-1, pump-2}. After {pump-2, pump-3}. Plan closes pump-1, opens pump-3, and keeps pump-2.

Invariant

An empty source never broadens to all devices. Fallback dashboards must be explicit nodes.

Shared demand
