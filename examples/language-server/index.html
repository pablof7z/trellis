Example 2

Mini language server

File contents derive a module graph, diagnostics, semantic tokens, watcher demand, and editor output frames.

file_contents
 -> module_graph
 -> diagnostics
 -> semantic_tokens
 -> editor_output #17

Problem

Deleted files must clear diagnostics. Import changes must invalidate dependent modules. Workspace close must stop watchers.

Trace

Remove `parser.rs`, recompute the module graph, close its watcher, clear diagnostics, and emit a semantic-token rebaseline.

Invariant

A diagnostic output frame cannot mention a file absent from the current module graph.

Oracle

Full parse of current files must produce the same diagnostics as incremental propagation.
