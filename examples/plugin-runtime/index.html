Example 4

Plugin runtime

Enabled plugins derive commands, file watchers, workers, and panels. Disabling a plugin should tear down everything it contributed.

enabled_plugins
 -> contributed_commands
 -> file_watchers
 -> workers
 -> panels #24

Problem

Manifest edits and plugin disables often leave stale commands, orphaned workers, or forgotten watchers.

Trace

Disable plugin `fmt`. Remove commands, close watchers, cancel workers, clear panels, and emit a shell revision.

Invariant

Every contributed resource is owned by the plugin scope that introduced it.

Audit

Dependency identity shows which manifest field produced each contribution.
