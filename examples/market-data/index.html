Market-data terminal

Watchlists derive quote streams, depth subscriptions, alert jobs, and revisioned screen output.

watchlist
 -> visible_symbols
 -> quote_and_depth_demand
 -> alert_jobs
 -> terminal_output #109

Problem

Symbol removals should close feeds and clear rows. Fast churn should not leave old depth subscriptions open.

Trace

Remove TSLA, add AMD. Plan closes TSLA quote/depth keys, opens AMD keys, and emits a terminal delta.

Invariant

Materialized output revision must match the latest committed visible-symbol set.

Benchmark pressure

High churn tests propagation cost, allocation, and full-recompute checking overhead.
