Examples

Proof that this is not Nostr-specific.

Each example starts with canonical input, derives a collection, computes structural diffs, produces resource plans, and emits revisioned output.

1. Local-first issue tracker

Workspace switches, permission revokes, offline replay, stale row clearing, and sync window teardown.

2. Mini language server

File deletion clears diagnostics, import changes invalidate dependents, and watcher scopes close.

3. MQTT fleet dashboard

Filter shrink unsubscribes topics, empty sets subscribe to nothing, and shared topics stay alive.

4. Plugin runtime

Plugin disable tears down commands, watchers, workers, and panels.

5. Nostr feed

Account switches, contact-list replacement, source shrink, and projection rebaseline.

6. Market-data terminal

Watchlist changes reconcile quote streams, depth subscriptions, and revisioned screen output.

7. Codebase Observatory

An interactive IDE workbench for stale diagnostics, watchers, host results, output frames, and replayable traces.

The examples are the API pressure test.
