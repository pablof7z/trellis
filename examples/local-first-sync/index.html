Local-first issue tracker

The active workspace determines visible projects, sync shapes, local replica demand, and the materialized issue board.

active_workspace
 -> visible_projects
 -> sync_shapes
 -> issue_rows
 -> issue_board #42

Problem

Workspace switches and permission revokes must close obsolete sync windows and clear stale rows.

Trace

Old workspace {A, B}. New workspace {B, C}. Plan closes A, opens C, keeps B, and emits a board revision.

Invariant

Permission shrink withdraws demand. Empty permission sets fail closed.

Oracle

Incremental board rows must match a full recompute from workspace, permissions, cache, and live scopes.
