Trellis vs Salsa

Salsa is a strong model for pure incremental queries. Trellis is for resource plans and materialized output lifecycles driven by derived collections.

Trellis

Best fit
Compilers, analyzers, pure derived facts.

A language server may need both: pure query recomputation and live watcher/output reconciliation.
