Comparisons

Neighboring tools solve neighboring problems.

The message is not that signals, Rx, Salsa, dataflow, or controllers are wrong. They usually stop at a different boundary.

Signals

Value invalidation and UI view updates.

Rx

Stream composition and event transforms.

Salsa

Pure incremental queries and memoized analysis.

Dataflow

Distributed or batch-oriented data movement.

Controller runtimes

Reconcile external desired state, often at a system boundary.

Use Trellis when a derived value owns external resources.
