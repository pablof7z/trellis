Trellis vs Rx

Rx is strong for composing event streams. Trellis is about reconciling desired resource ownership from derived graph state.

Question
Rx
Trellis
Main unit
Observable streams and operators.
Transactions, diffs, plans, scopes, revisions.
Lifecycle
Subscription disposal is usually host discipline.
Scoped teardown is a graph invariant.
Best fit
Event pipelines and async composition.
State-derived resource lifecycle and materialized outputs.

Trellis can sit beside streams. It decides what streams should exist.
