# kind:30315

The Status event in tenex-edge's NIP-29 wire codec (KIND_STATUS = 30315). Tags: d=session_id, title, status=busy|idle, host, repeated h=channel, optional slug/rel-cwd/expiration. Content = the live activity text (the 'doing X' string, empty when idle). A NIP-33 replaceable event addressed by (pubkey, d=session_id).
