# NodeLabels

A per-surface registry mapping graph-local NodeIds to stable semantic paths (e.g. status/<session>/activity, subscriptions/session/<id>/channels, hook/<session>/cursor), populated at node creation, so persisted receipts and commit ledgers carry human/agent-readable labels instead of meaningless integers like {"inputs":[2,3]}.
