# Flight Recorder (workstation)

Trellis's existing trace-replay UI (trellis#160), a full evidence workstation with provenance, transaction timeline, phase ladder, receipt inspector, and replay/raw-trace tabs; functionally a time-travel/trace-replay tool already shipping, distinct from NMP's nmp-devtools or Chirp's X-Ray pane.
