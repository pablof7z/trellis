# trellis_commits

An all-commit ledger table separate from artifact receipts — records every Trellis transaction per surface including no-ops and suppressed publishes, with trigger kind, changed inputs/derived/collections (as labels), command/output counts, duration, and graph size. The evidence source for 'is Trellis earning its keep' value metrics.
