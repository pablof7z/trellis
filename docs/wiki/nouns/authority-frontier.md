# authority frontier

An explicit per-surface classification along a five-mode ladder: imperative → shadow → advisory → authoritative → projection-owned — where each surface's mode is code-registered and CI-ratcheted, and promotion is gated by green conformance instruments (oracle, replay, acid, bypass-ratchet).
