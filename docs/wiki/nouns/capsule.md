# capsule

A recorded, replayable debugging artifact defined as trace + receipts with scrub/reconstruction capability (not re-execution); has its own envelope version embedding TRACE_FORMAT_VERSION, NMP receipt schema version, and producer metadata, plus a symbolication manifest and local/shareable redaction modes.
