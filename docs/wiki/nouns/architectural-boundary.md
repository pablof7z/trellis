# architectural boundary

Trellis owns decisions; the host owns observations and effects. The world's facts (ProcessExited, DistillCompleted, RelayPublishAccepted, ClockTick) enter as a single input journal; Trellis derives sessions/status/who/context/outbox/subscriptions. SQLite becomes a durable log + projection store.
