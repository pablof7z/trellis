# capsule (v1)

A recorded, replayable debugging artifact containing trace + receipts (not re-executable scripts) that can be scrubbed/reconstructed by an agent that never saw the session, requiring a symbolication manifest (NodeId/ScopeId → NMP-semantic-label registry) and its own envelope version independent of Trellis's TRACE_FORMAT_VERSION.
