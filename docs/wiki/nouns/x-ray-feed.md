# X-Ray Feed

A toggleable developer overlay on the real Chirp feed that shows live relay-subscription reconciliation with causal chains — subscriptions opening/replacing/closing per relay, each command annotated with its cause, scope teardown as a visible cascade, and shared-ownership refcounts.
