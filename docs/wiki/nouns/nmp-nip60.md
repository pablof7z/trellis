# nmp-nip60

The NMP crate owning pure NIP-60/61 mechanics: event codecs, Cashu proof/DLEQ/P2PK types, and validation. It has no I/O and no policy — backend selection, operation journal, and relay acquisition live in nmp-wallet, not here.
