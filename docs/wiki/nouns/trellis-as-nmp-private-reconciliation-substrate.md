# Trellis (as NMP private reconciliation substrate)

Private reconciliation machinery below NMP's typed read sessions; per ADR-0075, NMP may use Trellis as a private substrate but Trellis primitives must never become app/native/web-facing — NMP owns resource identity semantics even where Trellis owns identity mechanics.
