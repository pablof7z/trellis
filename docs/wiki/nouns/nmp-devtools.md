# nmp-devtools

A dev-build-only NMP crate exposing Trellis trace/audit data via an MCP server with query endpoints (why_subscription_open, why_view_stale, what_closed_this_relay, list_scope_inventory, replay_session) plus redacted trace export for bug reports; explicitly excludes metrics/dashboards/log aggregation — receipts only.
