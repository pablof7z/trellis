# X-Ray (X-Ray Feed)

A toggleable developer overlay on the real Chirp feed that shows live relay-subscription reconciliation with causal chains — subscriptions opening/replacing/closing per relay, each annotated with its cause, scope teardown cascades, and shared-ownership refcounts.
