# kind:30315 (KIND_STATUS)

The Nostr event kind for agent status: a NIP-33 replaceable event addressed by (pubkey, d=session_id) carrying tags for session id, title, busy/idle status, host, channel h-tags, slug, rel-cwd, and NIP-40 expiration, with content = the live 'doing X' activity text.
