Documentation starts with the model.

The crate API is still early. The durable contract is the category model: explicit nodes, structural diffs, resource plans, scopes, transactions, revisioned outputs, and full-recompute checks.

Module overview

graph

Graph construction, node identity, dependency edges, validation, and transaction entry points.

diff

Set and map structural diffs with deterministic ordering.

plan

ResourcePlan and plain command payloads returned to the host actor.

scope

Scope ownership, teardown, shared resource ownership, and output attachment.

output

MaterializedOutput frames, revisions, baselines, deltas, clears, and rebaselines.

testing

FullRecomputeOracle, invariant helpers, and property-test support.
